# Sobre o repositório
Esse repositório contém codigós de implementação de algoritmos para filtragem de imagens digitais, aplicando fundamentos matemáticos como transformada de Fourier e convolução.

# Linguagens e ferramentas utilizadas
- Python
- NumPy
- Matplotlib
- Jupyter Notebooks
- SciPy
- OpenCV

# Estrutura do repositório
- `notebooks/`: Contém notebooks Jupyter com exemplos e explicações detalhadas dos algoritmos implementados.
- `data/`: Pasta destinada a armazenar conjuntos de dados utilizados para testes e validações dos algoritmos. 
- `apresentacao.pdf`: Apresentação em PDF que resume os principais conceitos e resultados do projeto.
- `requirements.txt`: Lista de dependências necessárias para executar os códigos do repositório.
- `README.md`: Arquivo de documentação do repositório.

# Como usar
1. Clone o repositório.
2. Crie um ambiente virtual:
    ```bash
    python -m venv .venv
    source .venv/bin/activate  # No Windows use `.venv\Scripts\activate`
    ```
3. Instale as dependências:
    ```bash
    pip install -r requirements.txt
    ```
4. Navegue até a pasta `notebooks/` e abra os notebooks Jupyter para explorar os algoritmos implementados.
